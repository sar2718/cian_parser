## [Unreleased]

## [1.1.0] - 2026-05-11

### Добавлено
- Примеры данных и результатов анализа в папке `examples/`
- Полный датасет объявлений (unique.json) для демонстрации
- Статистика по районам Москвы (district_stats.json)
- Визуализации: гистограммы цен и боксплоты
- Подробный README с описанием данных и интерпретацией графиков
- Скрипт `generate_examples.py` для автоматической генерации примеров
- Дополнительные зависимости для модулей анализа в requirements.txt

### Изменено
- Обновлен .gitignore для исключения рабочих данных, но сохранения примеров
- Обновлен основной README с секцией примеров результатов

## [1.0.0] - 2026-05-10

### Добавлено
- Документация проекта (README.md)
- Лицензия MIT
- Расширенный .gitignore
- requirements-dev.txt для инструментов разработки
- Парсер списка объявлений с Cian.ru
- Парсер детальных страниц объявлений
- Система повторных попыток при ошибках сети
- Ротация User-Agent
- Разбиение на ценовые диапазоны для обхода лимитов пагинации
- Модули анализа данных (гистограммы, тепловые карты, боксплоты)
- Загрузка данных в ClickHouse
- Удаление дубликатов
- Анализ по районам

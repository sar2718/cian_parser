# Вклад в проект

Спасибо за интерес к проекту! Мы приветствуем любой вклад.

## Как внести вклад

### Сообщить об ошибке

Если вы нашли баг:

1. Проверьте, не создан ли уже Issue с этой проблемой
2. Создайте новый Issue с подробным описанием:
   - Шаги для воспроизведения
   - Ожидаемое поведение
   - Фактическое поведение
   - Версия Python и ОС
   - Логи ошибок (если есть)

### Предложить улучшение

Есть идея для новой функции?

1. Создайте Issue с описанием предложения
2. Объясните, какую проблему это решит
3. Дождитесь обсуждения перед началом работы

### Отправить Pull Request

1. Сделайте Fork репозитория
2. Создайте ветку для ваших изменений:
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. Внесите изменения и зафиксируйте их:
   ```bash
   git commit -m "Добавить amazing-feature"
   ```
4. Отправьте изменения в ваш Fork:
   ```bash
   git push origin feature/amazing-feature
   ```
5. Откройте Pull Request

## Стандарты кода

- Следуйте PEP 8
- Используйте понятные имена переменных и функций
- Добавляйте комментарии для сложной логики
- Обновляйте документацию при необходимости

## Тестирование

Перед отправкой PR убедитесь, что:

- Код работает корректно
- Не нарушена существующая функциональность
- Добавлены необходимые зависимости в requirements.txt

## Вопросы?

Не стесняйтесь создавать Issue с вопросами!
